
# PythonOS 
A open-source Python Console based OS with Linux like interface.

---

### Apps (Filename):
* Timer (timer.py)
* Calculator *Team Edition* (calculator.py)
* Chat *Only on Repl.it currently* (chat.py)
* Code Editor *Team Editon* (inspyre.py)
